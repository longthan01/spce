# SomeApp

Really simple test app for SPCE

## prerequisites
* dotNet core 5.x
* Nodejs 10.x
  
## How to run
### backend
on terminal 1
```bash
cd ./be/someapp
dotnet build
dotnet run
```
server should start on https://localhost:5001
### frontend
on terminal 2
```bash
cd ./fe/some-app
npm install
npm start
```
app should start on http://localhost:3000
